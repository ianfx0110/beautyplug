# BeautyPlug

BeautyPlug is a comprehensive beauty services management platform designed to streamline operations for beauty professionals and businesses. It enables stylists, salons, and beauty service providers to efficiently manage client appointments, maintain inventory of beauty products, track client interactions and preferences, and deliver a seamless booking experience through an intuitive web application.

The platform features a robust backend API and a modern frontend interface, providing both service providers and clients with the tools they need to schedule appointments, browse services, and manage their beauty care needs in one centralized location.

This repository contains the complete BeautyPlug project, including an Express-based API server and a Vite-powered frontend application.

## Structure

- `api/` - Express-based API server.
- `beautyplugtwo/` - Frontend application built with Vite.
- `database design/` - Database schema and documentation files.

## Getting Started

### API

1. Open a terminal and go to `api/`:
   ```bash
   cd api
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the API server:
   ```bash
   npm start
   ```

### Frontend

1. Open another terminal and go to `beautyplugtwo/`:
   ```bash
   cd beautyplugtwo
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the frontend:
   ```bash
   npm run dev
   ```

## Notes

- The API server runs from `api/app.js`.
- The frontend application entry point is `beautyplugtwo/src/main.jsx`.
- Database documentation and schema files are available under `database design/`.
